description: Getting started if documentation is missing.
---

## Symptoms

- You log in but see an empty dashboard
- Your previously created docs seem gone

## Quick Fix

<Steps>
  <Step title="Check project selection">
    Ensure you have the correct project chosen from the sidebar or project menu.
  </Step>
  <Step title="Refresh the page">
    Sometimes a simple refresh resolves missing documents.
  </Step>
  <Step title="Contact support">
    If docs are still missing, email support@ilyaionov.com.
  </Step>
</Steps>
