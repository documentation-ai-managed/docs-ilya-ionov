## Upgrade Instructions

Follow these steps to upgrade Ilya Ionov to the latest version.

<Steps>
  <Step title="Backup Your Site" icon="database">

    Create a backup of your current documentation.

````bash
cp -r docs/ docs-backup-$(date +%Y%m%d)
````

  </Step>
  <Step title="Update Package" icon="package">

    Install the latest version using your package manager.

    <CodeGroup tabs="npm,yarn">
````bash
npm install ilya-ionov@latest
````
````bash
yarn add ilya-ionov@latest
````
    </CodeGroup>

  </Step>
  <Step title="Run Migration" icon="settings">

    Execute the migration script if upgrading from v0.x.

````bash
npx ilya-ionov migrate
````

  </Step>
  <Step title="Deploy" icon="rocket">

    Build and deploy your updated site.

````bash
npm run build
npm run deploy
````

  </Step>
</Steps>

<Callout kind="tip">
  Review the [migration guide](/migration) for detailed changes between versions.
</Callout>
