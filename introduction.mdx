## Overview

Ilya Ionov provides a centralized space for you to organize, create, and manage all your project documentation. You can build comprehensive docs sites using MDX, collaborate with teams, and deploy updates seamlessly. Start by exploring key features and follow the quick start guide to set up your first documentation page.

## Key Features

Ilya Ionov simplifies documentation management with powerful tools tailored for developers and teams.

## Quick Start

Get your documentation space running in minutes. Follow these steps to create your first page.

<Steps>
  <Step title="Create a New Doc" icon="plus">
    Navigate to your dashboard and select "New Page". Choose the `introduction.mdx` template.

    <CodeGroup tabs="CLI,Web UI">
      ```bash
      ilya init my-docs
      cd my-docs
      ilya pages create introduction.mdx
      ```
      ```typescript
      // Web UI equivalent via API
      await fetch('https://api.example.com/docs/pages', {
        method: 'POST',
        body: JSON.stringify({ title: 'Introduction', file: 'introduction.mdx' })
      });
      ```
    </CodeGroup>
  </Step>

  <Step title="Add Content" icon="edit-3">
    Edit the YAML frontmatter and add H2 sections. Use components for rich layouts.

    ```mdx
    ---
    title: My Page
    description: Page summary.
    ---
    
    ## Start Here
    
    Welcome to your docs.
    ```
  </Step>

  <Step title="Preview and Deploy" icon="eye">
    Preview changes live, then deploy with one click.

    <Callout kind="tip">
      Use `ilya deploy` in your terminal for production-ready builds.
    </Callout>
  </Step>
</Steps>

## Brand Guidelines

Follow these guidelines to maintain consistency across your documentation.

<Callout kind="info" style={{ '--brand-color': '#3B82F6' }}>
  Your primary brand color is `#3B82F6`. Apply it to headings, buttons, and accents. Avoid using it for text to ensure readability.
</Callout>

<Tabs>
  <Tab title="Primary" icon="circle">
    - Color: `#3B82F6`
    - Usage: Buttons, links, highlights
  </Tab>
  <Tab title="Secondary" icon="circle">
    - Colors: `#1E40AF`, `#60A5FA`
    - Usage: Subtle accents, backgrounds
  </Tab>
</Tabs>
